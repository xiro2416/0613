# 模块四：股市圈层社交辅助模块

> 核心定位：情境感知型关系策动器、同频共鸣发生器

## 接口状态 ✅

- 文件：`backend/modules/social_tactician.py`
- 类：`SocialTactician(BaseModule)`
- 模块名：`social_tactician`
- 方法：`process(context: ModuleContext) -> ModuleResult`

## 人设路由策略

| 人设 | 激活方式 | 话术风格 |
|---|---|---|
| GENTLE | 可触发 | 温暖鼓励型 |
| RATIONAL | 可触发 | 专业分析型话题 |
| MEME | 可触发 | 圈层热梗幽默话术 |

## 核心职责（PRD 定义）

1. 打造市况+持仓感知型恋爱军师
2. 依托实时行情、标的涨跌、市场情绪生成专属社交话术
3. 感知社交对象持仓、板块行情波动
4. 生成低刻意度开场白、专业底层逻辑话题、圈层热梗幽默话术
5. 借盘面聊认知、聊情绪、聊人生，辅助社交破冰
